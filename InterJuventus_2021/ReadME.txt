GUIDA OPERATIVA

Nella cartella principale denominata SMA_Fossati_Pretini, oltre al report, si troverà una cartella chiamata Codice_Dati, all’interno della quale si troverà:
-	interjuve.csv: il dataset di riferimento per le analisi
-	Raccolta_Dati.ipynb : jupyter notebook che contiene il codice di raccolta dei dati da Twitter, con il quale si è prodotto il file interjuve.csv
-	SNA.ipynb : jupyter notebook contenente il codice inerente alla Social Network Analysis
-	sentiment_analysis.ipynb : jupyter notebook contenente il codice inerente alla Sentiment Analysis 
-	sentiment_emoji.ipynb: jupyter notebook per la gestione delle emoji
